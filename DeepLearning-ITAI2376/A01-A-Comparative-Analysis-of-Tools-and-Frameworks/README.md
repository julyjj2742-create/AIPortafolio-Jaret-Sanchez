# A01 - A Comparative Analysis of Machine Learning and Deep Learning Tools and Frameworks

## Overview
In this assignment, I compared two of the most popular deep learning frameworks: TensorFlow and PyTorch. I looked at where each one came from, what they’re mainly used for, and the features that make them different. As I explained in my paper, “TensorFlow was developed by Google and released in 2015… allowing us to build and deploy machine learning models,” while “PyTorch was developed by Meta and released in 2016… making experimentations for researchers easier.”

I also included simple neural network examples for both frameworks to show how their code structure and workflow differ in practice.

## Purpose
The main goal of this assignment was to understand how TensorFlow and PyTorch compare when it comes to flexibility, ease of use, deployment options, and real‑world applications. This helped me get a better sense of when one framework might be a better choice than the other.

## Key Concepts
### TensorFlow:
- Scales well across different platforms
- Has a big ecosystem (Keras, TF Lite, TF.js, etc.)
- Supports automatic differentiation
- Works with multiple languages
- Includes tools for model optimization

### PyTorch:
- Works smoothly with Python libraries
- Uses dynamic computation graphs
- Supports automatic differentiation
- Has strong CUDA and multiprocessing support
