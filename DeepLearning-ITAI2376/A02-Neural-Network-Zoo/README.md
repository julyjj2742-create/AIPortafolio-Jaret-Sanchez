# A02 - Neural Network Zoo

## Overview
In this assignment, I explored different types of neural networks and explained how each one works by representing them as animals. Using animals to represent each network helped me explain the concepts in a more creative way.

## Purpose
The purpose of this assignment was to build a clear understanding of how different neural networks operate and what makes each one unique. By comparing their structures, strengths, limitations, and real‑world applications, I learned why certain networks are better suited for specific tasks.

## Key Concepts
### CNNs (Cheetah)
- Great for image tasks
- Use convolution and pooling layers
- Learn visual patterns
- Strengths: fast, accurate, strong at spatial features
- Limitations: need large datasets, expensive to train
- Cheetah represents speed and strong vision

### RNNs (Raccoon)
- Process data one step at a time
- Use feedback loops to remember previous inputs
- Strengths: good for sequential data
- Limitations: struggle with long‑term memory
- Raccoon represents memory and problem‑solving

### LSRMs (Lemur)
- Special type of RNN with gates
- Designed to remember long‑term information
- Strengths: handle long sequences, avoid vanishing gradients
- Limitations: more complex and slower
- Lemur represents alertness and long‑term memory

### GANs (Fox)
- Two networks: generator and discriminator
- Compete to create realistic synthetic data
- Strengths: powerful for image generation and creative tasks
- Limitations: hard to train and stabilize
- Fox represents cleverness and strategy
