# A03 - Deep Learning for an 11-year-old

## Overview
In this assignment, I created a kid‑friendly video explaining what gradient descent is. I used a simple story with a mountain (The Mountain of Mistakes), a small robot (RoboKid), and a teacher robot (Professor Pixel) to make the concept easier to understand. Instead of focusing on math, I explained gradient descent as a process of taking small steps downhill to reduce mistakes, similar to how a robot would try to find the lowest point on a mountain. This helped turn a complex deep learning idea into something an 11‑year‑old could follow.

## Purpose
The purpose of this assignment was to explain gradient descent in a way that is clear, simple, and engaging for a younger audience. By using characters and a visual metaphor, I learned how to break down a technical concept into a story that still keeps the main idea accurate. This assignment also helped me practice communicating AI topics in a creative way, which is an important skill when teaching or presenting AI concepts to people who are new to them.
